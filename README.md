# Oniguruma++

## What's this?

This is a C++ wrapper of the Oniguruma regular expression engine.

## Usage

```bash
git clone https://github.com/katahiromz/onigpp
git submodule update --init --recursive
cd onigpp
mkdir build
cmake -B build
cmake --build build
```
